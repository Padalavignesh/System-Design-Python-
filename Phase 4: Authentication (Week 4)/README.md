# 🔐 Phase 4: Authentication (Week 4)

A structured roadmap to learn **Authentication** and **Authorization** for modern backend applications using **Python**, **Django**, and **Django REST Framework (DRF)**.

---

# 📚 Learning Objectives

During this phase, you will learn the following authentication concepts:

- JWT (JSON Web Token)
- Refresh Token
- OAuth
- Password Hashing
- Role-Based Access Control (RBAC)

---

# 🛠 Implement

Build a complete authentication system with the following features:

## 👤 User Signup

- Register a new user
- Validate user input
- Store passwords securely using hashing

---

## 🔑 User Login

- Authenticate users
- Generate JWT Access Token
- Generate Refresh Token
- Secure protected API endpoints

---

## 🔄 Forgot Password

- Request password reset
- Generate secure reset token
- Reset password using the token

---

## 📧 Email Verification

- Send verification email after registration
- Verify user account using a unique token
- Restrict login until email is verified

- Implement Role-Based Access Control (RBAC).
