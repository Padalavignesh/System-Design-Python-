# 🚀 Phase 1: Backend Fundamentals (Week 1)

A structured roadmap to learn the core backend concepts using **Python**, **Django**, and **Django REST Framework (DRF)**.

---

# 📚 Learning Objectives

During this phase, you will learn the following backend fundamentals:

- HTTP & HTTPS
- REST APIs
- CRUD Operations
- JSON
- HTTP Status Codes
- Authentication
- Authorization
- Cookies vs JWT
- Sessions
- CORS (Cross-Origin Resource Sharing)

---

Features:

- Delete Notes

## ✅ Todo API

Features:

- Create Todo
- Read Todos
- Update Todo
- Delete Todo

## 🔐 Authentication API

Features:

- User Registration
- User Login
- User Logout
- User Profile
