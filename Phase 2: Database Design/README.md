
# 🗄️ Phase 2: Database Design (Week 2)

A complete roadmap to learn **Database Design** for Backend Development using **PostgreSQL** and **MySQL**.

---

# 📚 Learning Objectives

During this phase, you will learn the following database concepts:

- SQL Basics
- Primary Key
- Foreign Key
- Indexing
- Joins
- Constraints
- Transactions
- ACID Properties
- Normalization
- Denormalization

# 🎯 Expected Outcome

After completing this phase, you will be able to:

- Write SQL queries confidently.
- Design normalized relational databases.
- Create tables with proper relationships.
- Use Primary Keys and Foreign Keys effectively.
- Perform INNER, LEFT, RIGHT, and FULL JOIN operations.
- Improve query performance using indexes.
- Apply database constraints for data integrity.
- Understand and use transactions.
- Explain ACID properties.
- Decide when to normalize or denormalize a database.
- Design real-world relational databases.
